# Retro Theming

Use this skill when creating or modifying visual components, CSS, or layout elements.

## Requirements

- Use CSS Variables from `src/styles/tokens.css` for all colors, spacing, and fonts
- Use dotted/dashed borders for containers and decorative elements
- Monospace font stack: `'JetBrains Mono', 'IBM Plex Mono', monospace`
- Apply scanline overlay on images where applicable
- Use flat colors from the blue-green palette only
- Zero border-radius (or max 4px)
- ASCII-art decorations for dividers and section breaks
- No `backdrop-filter: blur()`
- No Tailwind CSS utility classes
- No large shadows or gradients

## Color Palette

- `--color-cyan: #00ffcc`
- `--color-teal: #00ccff`
- `--color-blue: #0066ff`
- `--color-blue-deep: #003366`
- `--color-accent: #ffff00`
